# 主讲教师代表性教学获奖成果（原件扫描件）

> 用途：仅供管理员审核，不对评审专家公开
> 格式：1个PDF文件
> 要求：与匿名版对应的获奖证书/文件的原件扫描件

---

## 准备步骤

1. **收集证书原件**：找到匿名版中填写的每项获奖对应的证书/文件原件
2. **扫描或拍照**：每份证书单独扫描/拍照，确保文字清晰可辨
3. **按顺序排列**：与匿名版填写顺序一致（第1项→第2项→...）
4. **合并为1个PDF**：使用"合并PDF"工具将所有扫描件合并为1个文件
5. **上传**：上传至大赛系统"原件扫描件"上传口

## PDF内容结构

```
第1页：第1项获奖证书扫描件
第2页：第2项获奖证书扫描件
第3页：第3项获奖证书扫描件
...（最多5页，对应最多5项）
```

## 注意事项

- 此版本包含真实姓名和单位信息（不匿名）
- 每项获奖必须有对应的证书/证明文件
- 扫描件应清晰可读，包含完整的证书内容（获奖人、颁奖单位、日期、公章）
- 如证书为电子版，可直接截图或导出
- PDF总大小建议不超过 20MB
