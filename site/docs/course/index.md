# 课程概览

## 课程基本信息

| 项目 | 内容 |
|---|---|
| 课程名称 | Java Web应用开发 |
| 课程性质 | 专业核心课 |
| 授课对象 | 计算机科学与技术/软件工程本科生 |
| 教学目标 | 支撑Web应用开发、工程规范与安全治理能力 |
| 赛道定位 | 人工智能赋能教学创新（课前-课中-课后闭环） |

## 课程目标

1. 掌握 Servlet、JSP、Filter、Listener 与三层架构。
2. 理解 Authentication、Authorization、RBAC 等关键安全机制。
3. 能完成“开发-测试-审查-迭代”的项目化实践。
4. 建立“最小权限、默认拒绝、验证驱动”的工程思维。

## 16讲安排与AI应用标记

| 讲次 | 主题 | AI应用 |
|---|---|---|
| 第1-5讲 | Web基础、Servlet、JSP、会话管理 | 基础阶段 |
| 第6-10讲 | MVC、JDBC、DAO、Filter、Listener | AI辅助代码理解与审查 |
| 第11-14讲 | 文件处理、异步交互、项目实战 | 个性化任务与审查联动 |
| **第15讲** | **用户认证与权限控制** | **核心案例：AI审查+验证闭环** |
| 第16讲 | 工程化部署与课程总结 | AI辅助复盘与改进建议 |
