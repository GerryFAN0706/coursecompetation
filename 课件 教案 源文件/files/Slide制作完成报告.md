## 📖 使用说明

### 如何预览Slide

**使用VS Code + Marp插件**:
1. 安装Marp for VS Code插件
2. 打开任意`.md`文件
3. 点击右上角"Open Preview"图标
4. 预览PPT效果

### 如何导出PDF

**方法一：VS Code导出**:
1. 预览Slide
2. 点击右上角"Export Slide Deck"
3. 选择PDF格式
4. 设置分辨率为1600×900

**方法二：Marp CLI**:
```bash
marp --pdf --allow-local-files 第一讲_Web核心脉络_0511.md
```

### 如何修改

**修改内容**:
- 直接编辑`.md`文件即可
- 保持格式规范（参考《PPT格式规范说明.md》）

**修改样式**:
- 不建议修改，已优化至最佳
- 如需调整Logo位置，修改YAML中的style部分
