# 人才培养成果证明材料（原件扫描件）

> 用途：仅供管理员审核，不对评审专家公开
> 格式：1个PDF文件
> 要求：与匿名版对应的成果证明文件的原件扫描件

---

## 准备步骤

1. **收集证明原件**：找到匿名版中填写的每项成果对应的证明文件
2. **扫描或拍照**：每份证明单独扫描/拍照，确保文字清晰可辨
3. **按顺序排列**：与匿名版填写顺序一致
4. **合并为1个PDF**：所有扫描件合并为1个文件
5. **上传**：上传至大赛系统"原件扫描件"上传口

## 各类成果的证明文件

| 成果类型 | 需要的证明文件 |
|:---|:---|
| 学科竞赛获奖 | 获奖证书扫描件（需含学生姓名、指导教师、奖项等级） |
| 大创项目 | 立项通知书 + 结题证书 |
| 学生论文 | 论文首页（含作者信息和期刊名）或录用通知 |
| 优秀毕业论文 | 评选结果文件或证书 |
| 创新创业大赛 | 获奖证书或官方公示文件截图 |
| 技术认证 | 认证证书扫描件 |

## 注意事项

- 此版本包含真实姓名和单位信息（不匿名）
- 每项成果必须有对应的证明文件
- 如果学生姓名出现在证书上，保留原始即可
- 指导教师信息保留原始
- PDF总大小建议不超过 20MB
