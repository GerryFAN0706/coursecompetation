## 常见问题

<div class="info-box blue">
  <span class="icon">❓</span>
  <div><strong>Q：AI会不会替代学生思考？</strong><br/>A：AI是"辅助诊断者"而非"答案提供者"。提示词设计要求AI只输出问题位置和风险说明，不直接给出修复代码。学生需要自行理解问题并编写修复方案。教师在过程中进行知识点纠偏，确保学习深度。</div>
</div>

<div class="info-box blue">
  <span class="icon">❓</span>
  <div><strong>Q：成效数据的统计口径是什么？</strong><br/>A：实验组N=58 vs 对照组N=62，同教师同课程同教材，16周准实验设计。详细统计方法、数据采集过程和分析口径见<a href="/coursecompetation/results/">成效与数据</a>页面。</div>
</div>

<div class="info-box blue">
  <span class="icon">❓</span>
  <div><strong>Q：这套模式对教师技术门槛高吗？</strong><br/>A：最低要求仅需"能使用一种生成式AI工具"。我们提供了10条可直接复用的提示词模板和详细的使用说明。三步迁移法支持渐进式采纳，无需一次性全面改造。</div>
</div>
