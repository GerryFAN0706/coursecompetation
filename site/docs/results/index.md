## 8) 测量方法、边界与声明

### 测量方法（摘要）
1. 编译通过率：Gitea首提编译状态自动统计。  
2. 功能完整性：按功能点清单逐项评分（0/0.5/1）。  
3. 安全漏洞数：教师+AI协同审查，按OWASP简化清单计数。  
4. 异常处理覆盖率：静态统计Checked Exception调用点覆盖。

### 可比性与控制
- 入学成绩可比：`t(118)=0.74, p=0.46, d=0.14`（无显著差异）
- 同课程、同教师、同教材、同学时、同实验环境
- 仅教学方式变量不同（是否引入AI辅助）

### 局限性
1. 样本量小于100，外推需谨慎。  
2. 准实验设计，存在年级效应/历史效应干扰。  
3. 学生知晓试点可能产生投入提升（霍桑效应）。  
4. 学生AI使用强度差异未纳入协变量。  
5. 教师经验自然增长可能带来附加提升。
