## 规范的包结构

###### Standard Package Structure

**项目目录 (Project Directory)**:

```
src/com/tyust/
  ├── controller/        ← Servlet控制器
  ├── service/           ← 业务逻辑层
  ├── dao/               ← 数据访问层
  ├── model/             ← JavaBean实体
  └── util/              ← 工具类(DBUtil)

webapp/
  ├── jsp/               ← JSP页面
  ├── WEB-INF/
  │   ├── web.xml
  │   └── lib/           ← JAR包
  └── css/               ← 样式文件
```

**原则**: 未写代码，结构先行  
*Principle: Structure first, code later*

## 分层架构

###### Layered Architecture

**四层结构 (Four-Layer Structure)**:

<!-- _class: cols2_ol_ci -->

- **Controller层** Servlet接收请求  
  *Receive requests*
- **Service层** 业务逻辑处理  
  *Business logic*
- **DAO层** 数据库操作  
  *Database operations*
- **Model层** JavaBean数据封装  
  *Data encapsulation*

**依赖关系 (Dependencies)**:
```
Controller → Service → DAO → Database
     ↓
   View (JSP)
```

## 第一站：Model层

###### Station 1: Model Layer

**定义数据模型** *Define data model*

```java
// Book.java - 图书JavaBean
public class Book implements Serializable {
    private Integer id;
    private String name;
    private String author;
    private Double price;
    
    // 构造函数、Getter/Setter省略
}
```

**作用 (Role)**:
- 📦 封装图书数据 *Encapsulate book data*
- 🔄 在各层间传递 *Transfer between layers*

## 第二站：DAO层

###### Station 2: DAO Layer

**数据访问对象** *Data Access Object*

```java
// BookDao.java
public class BookDao {
    public List<Book> findAll() {
        List<Book> list = new ArrayList<>();
        Connection conn = DBUtil.getConnection();
        // JDBC查询...
        return list;
    }
}
```

**职责 (Responsibilities)**:
- 🗄️ 连接数据库 *Connect database*
- 📥 查询数据 *Query data*
- 📦 封装成JavaBean *Wrap to JavaBean*

## 第三站：Service层

###### Station 3: Service Layer

**业务逻辑层** *Business Logic Layer*

```java
// BookService.java
public class BookService {
    private BookDao dao = new BookDao();
    
    public List<Book> getAllBooks() {
        // 可添加业务逻辑、缓存、验证等
        return dao.findAll();
    }
}
```

**职责 (Responsibilities)**:
- ⚙️ 业务逻辑处理 *Business logic*
- 🔐 权限验证 *Authorization*
- 📊 数据处理 *Data processing*

## 第四站：Controller层

###### Station 4: Controller Layer

**Servlet控制器** *Servlet Controller*

```java
@WebServlet("/listBooks")
public class BookListServlet extends HttpServlet {
    private BookService service = new BookService();
    
    protected void doGet(HttpServletRequest request, ...) {
        // 1. 调用Service
        List<Book> books = service.getAllBooks();
        
        // 2. 存入request
        request.setAttribute("bookList", books);
        
        // 3. 转发到JSP
        request.getRequestDispatcher("/jsp/bookList.jsp")
            .forward(request, response);
    }
}
```

## 第五站：View层

###### Station 5: View Layer

**JSP视图** *JSP View*

```jsp
<%@ taglib prefix="c" uri="http://java.sun.com/jsp/jstl/core" %>
<html>
<body>
    <h1>图书列表 (Book List)</h1>
    <table border="1">
        <tr><th>ID</th><th>书名</th><th>作者</th></tr>
        <c:forEach items="${bookList}" var="book">
            <tr>
                <td>${book.id}</td>
                <td>${book.name}</td>
                <td>${book.author}</td>
            </tr>
        </c:forEach>
    </table>
</body>
</html>
```

## 完整数据流

###### Complete Data Flow

**五站式旅程 (Five-Station Journey)**:

```
用户访问 /listBooks
  ↓
① Controller (BookListServlet)
  ↓
② Service (BookService.getAllBooks())
  ↓
③ DAO (BookDao.findAll()) → JDBC查询数据库
  ↓
④ 返回 List<Book> → setAttribute到request
  ↓
⑤ forward到 bookList.jsp → EL/JSTL显示
  ↓
HTML响应 → 浏览器显示表格
```
