# 课堂AI使用场景记录

> **课程**：Java Web应用开发（2025年秋季学期）
> **说明**：以下为课堂中学生使用AI工具的典型场景记录，照片需做面部模糊处理后使用。

---

## 场景一：AI辅助分组代码审查（第15讲）

### 场景描述
学生分为A/B/C三组，每组使用AI工具审查不同代码模块。学生在IDE中打开代码文件，将任务卡中的结构化提示词输入AI工具，阅读并讨论AI的审查输出。

### 照片拍摄建议
1. **全景照**：教室全景，学生分组坐在电脑前，屏幕上显示IDE和AI对话窗口
2. **小组讨论照**：3-4名学生围在一台电脑前，指着屏幕讨论AI输出
3. **AI对话截图**：学生输入提示词后AI返回的审查结果界面（需遮盖个人信息）
4. **教师巡视照**：教师站在学生旁边，指着屏幕上的AI输出进行指导

       AI返回了权限过滤器仅检查登录状态未检查角色的问题诊断。
       学生正在讨论是否采纳AI的修改建议。

## 场景二：AI辅助实验调试（第10讲MVC综合实战）

### 场景描述
学生在MVC综合实验中遇到Servlet转发与重定向的混淆问题。使用Cursor的AI功能描述错误现象，AI帮助定位问题原因（forward后未return导致重复响应），学生理解后自行修改代码。

### 照片标注模板
```
[照片编号] P10-01
[场景] 第10讲实验 - 学生使用AI辅助调试MVC项目
[时间] 2025年10月XX日 第7-8节
[说明] 学生遇到"IllegalStateException: Response already committed"错误，
       将错误信息输入AI工具，AI解释了forward后未return导致
       resp.sendRedirect被二次调用的问题。学生理解后自行添加return语句。
[隐私处理] 已对学生面部进行模糊处理
```

## 场景三：AI辅助个性化实验设计（教师端）

### 场景描述
教师使用AI工具为不同水平的学生生成个性化实验任务。基于学生前序作业的成绩表现，AI生成基础/进阶/挑战三个难度层级的实验要求。

### 照片标注模板
```
[照片编号] PT-01
[场景] 教师使用AI生成个性化实验任务
[时间] 2025年11月XX日 课前准备
[说明] 教师向AI描述课程进度和学生水平分布，
       AI生成三个难度层级的权限控制实验任务：
       基础（BCrypt升级）、进阶（三级权限）、挑战（Session安全增强）。
       教师在AI输出基础上调整措辞和要求后发布。
[隐私处理] 无学生信息
```
