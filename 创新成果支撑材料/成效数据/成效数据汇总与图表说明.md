# 成效数据汇总与图表说明

> **数据来源**：2025年秋季学期"Java Web应用开发"课程，2个教学班
> **样本量**：N=58（计科2301班30人，计科2302班28人）
> **对比基准**：2024年秋季学期同课程（未使用AI辅助），N=62
> **测量周期**：2025年9月—2025年12月（16周教学周期）
> **数据采集方**：任课教师 + 教学管理系统自动采集

## 七、数据注释与声明

### 样本信息
- 实验组（2025秋）：计科2301班30人 + 计科2302班28人 = 58人
- 对照组（2024秋）：计科2201班32人 + 计科2202班30人 = 62人
- 两组学生入学成绩（高考数学+英语）无显著差异（t检验 p=0.73）

### 对比基准
- 同一课程、同一教师、同一教材、同一实验环境
- 唯一变量：是否引入AI辅助教学

### 数据局限性
- 样本量较小（N<100），结论的推广性有限
- 未进行随机分组，存在年级效应干扰的可能
- 部分数据（如调试时间）依赖学生自报或平台记录，可能存在偏差
- AI使用程度因学生而异，未控制AI使用频率变量

### 图表制作建议
- 推荐工具：Excel / Python matplotlib / Echarts
- 配色：蓝色=对照组(2024)，绿色=实验组(2025)
- 每张图标注：样本量N、时间段、p值（如适用）
- 导出格式：PNG 300dpi，用于报告插图和网站展示
