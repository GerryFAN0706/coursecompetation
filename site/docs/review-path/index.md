# 评审路径

## 5分钟快速路径（建议现场优先）

1. [首页](/)：确认创新主线与关键指标。
2. [AI教学创新设计](/innovation/)：确认“问题-目标-路径”逻辑是否成立。
3. [核心案例：第15讲](/cases/lesson-15/)：确认课堂实施闭环是否真实可验证。
4. [成效与数据](/results/)：确认量化结果与证据口径。

## 15分钟完整路径（材料复核）

1. [课程概览](/course/)：课程目标、16讲映射与工具链。
2. [AI教学创新设计](/innovation/)：课前-课中-课后架构与人机协同。
3. [核心案例：第15讲](/cases/lesson-15/)：问题工程、课堂脚本、验证路径。
4. [成效与数据](/results/)：对比分析、方法口径、指标追溯。
5. [教学资源](/resources/)：提示词、测试清单、文档下载、证据索引。
6. [推广与复用](/promotion/)：实施条件与可迁移模式。

## 评审重点对照

| 评审关注点 | 推荐页面 |
|---|---|
| 是否有真实教学问题 | `/innovation/` |
| 是否体现人机协同教学 | `/cases/lesson-15/` |
| 是否有可验证成效 | `/results/` |
| 是否具备可复制推广价值 | `/promotion/` |
| 是否有完整支撑材料 | `/resources/` + `/evidence-index` |

## 一句话总览

本项目核心是将 AI 从“工具展示”升级为“教学闭环能力”，实现从课前设计到课后干预的可证据化改革。
