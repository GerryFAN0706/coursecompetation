## 设计要求

- **尺寸**: 1920 x 1080 像素（16:9）
- **风格**: 简洁学术，蓝色系主色调（#4472C4）
- **必须包含**:
  - 课程名称：`Java Web应用开发`
  - 副标题：`五维AI融合创新教学体系`
  - 四个AI工具图标/名称（体现创新特色）
  - 赛道标识：`第六届全国高校教师教学创新大赛 · 人工智能赛道`
- **不得包含**:
  - 教师姓名
  - 学校名称
  - 学院名称
  - 赛区名称
  - 任何可识别身份的信息

## 建议配色

| 元素 | 颜色 | 说明 |
|:---|:---|:---|
| 背景 | 渐变蓝 `#1a2a6c → #4472C4 → #6a94d4` | 与课件风格统一 |
| 标题文字 | 白色 `#FFFFFF` | 醒目 |
| 副标题 | 浅蓝 `#B0C4DE` | 层次分明 |
| WebDev助教 | 蓝色 `#4472C4` | 一致品牌色 |
| WebSec挑战 | 青色 `#00BCD4` | 安全主题 |
| TeamCoach | 绿色 `#4CAF50` | 协作主题 |
| Code Review | 紫色 `#7c3aed` | 对战主题 |

## 制作方式建议

1. **PowerPoint/Keynote**: 新建16:9幻灯片，设计后导出为PNG
2. **Canva**: 使用学术演示模板修改
3. **Figma**: 自由设计
4. 导出为 PNG 或 JPG，分辨率不低于 1920x1080
