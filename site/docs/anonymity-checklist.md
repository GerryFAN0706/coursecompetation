# 匿名化检查清单

## 发布模式

- `internal`：原始材料版本（校内留档）
- `public`：匿名评审版本（专家评审）

## 必检项

1. 教师姓名、联系方式、工号、学号是否暴露。
2. 学校名称、学院名称、校徽院徽是否符合匿名要求。
3. 平台账号昵称、浏览器收藏夹、系统页眉机构信息是否打码。
4. 学生照片是否已做人脸模糊。
5. 文档元数据（PDF作者、Word修订）是否清理。

## 指标可信度必检项

1. 每个关键指标是否可追溯到证据文档或数据文件。
2. 是否注明样本范围、统计周期和计算规则。
3. 是否区分“课堂观察数据”和“平台日志数据”。

## 发布前命令

```bash
npm run validate
npm run docs:build
```
