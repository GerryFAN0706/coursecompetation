## 说明

本文件夹存放与 `05_AI赛道证明材料_匿名版` **完全相同的截图**，区别在于：

| 对比项 | 匿名版（05） | 原件版（06） |
|:---|:---|:---|
| 教师姓名 | 遮盖为XXX | **保留原始** |
| 学校名称 | 遮盖为XXX大学 | **保留原始** |
| 学院名称 | 遮盖为XXX学院 | **保留原始** |
| 学生姓名/学号 | 遮盖 | **保留原始** |
| 域名/URL | 替换为example.edu.cn | **保留原始** |
| 截图数量 | 相同 | 相同 |
| 截图排列顺序 | 相同 | 相同 |

## 操作步骤

1. 先制作匿名版（05文件夹），确认截图内容完整
2. 将同一批截图的**原始未遮盖版本**按相同顺序排列
3. 合并为1个PDF文件
4. 上传至大赛系统
