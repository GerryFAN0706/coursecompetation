# B组任务卡：授权组（Authorization）

## 你的任务

审查系统的 **角色权限过滤器**，判断授权控制是否真正有效。

## 审查文件

1. `RoleBasedAuthFilter.java` — 角色权限过滤器

## 关注问题

1. 过滤器是否只判断了"已登录"，而没有判断角色？
2. 过滤器是否正确识别了 ADMIN 用户？
3. 普通用户（USER）是否仍然可以访问 `/admin/*` 路径？
4. 未登录用户访问是否被正确拦截？
5. 授权逻辑是否存在过宽或判断遗漏？

## 对应知识点

- **Authorization**（授权）：你能做什么？
- **RBAC**（基于角色的访问控制）：不同角色，不同权限
- **Filter**（过滤器）：统一控制受保护路径的入口
