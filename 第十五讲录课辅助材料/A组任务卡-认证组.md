# A组任务卡：认证组（Authentication）

## 你的任务

审查系统的 **登录与登出模块**，判断用户认证流程是否安全、规范。

## 审查文件

1. `LoginServlet.java` — 登录处理
2. `LogoutServlet.java` — 登出处理

## 关注问题

1. 登录成功后，用户信息是否正确保存到 Session？
2. 登录失败时，系统是否给出了合理反馈？
3. 登出时，是否真正清除了所有登录状态？
4. 登出后，是否还存在会话残留问题？
5. 重新登录后，状态是否正常？

## AI 审查提示词

将以下内容复制到 AI 工具中，并附上 `LoginServlet.java` 和 `LogoutServlet.java` 的代码：

```
请审查这个 Java Web 登录与登出模块，重点检查：

1. 登录成功后是否正确将用户信息保存到 Session
2. 登录失败时是否进行了合理反馈
3. 注销时是否真正清除了登录状态
4. 是否存在会话残留问题
5. 是否存在用户重新登录后状态混乱的问题

请按以下格式输出：
- 问题位置
- 风险说明
- 对应知识点（Authentication / Session-Based Login / Logout）
- 修改建议
- 建议验证方式
```
