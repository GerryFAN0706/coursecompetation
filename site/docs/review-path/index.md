## 🕐 15 分钟完整评审路径

适用于材料复核、深入评议等需要全面了解的场景，覆盖从课程背景到推广价值的完整链条。

<div class="timeline">
  <div class="timeline-item">
    <h4>第 1–2 分钟：课程全貌</h4>
    <p>访问 <a href="/course/">课程概览</a>，了解课程目标、16 讲内容映射与 AI 工具链全景，建立课程背景认知。</p>
  </div>
  <div class="timeline-item">
    <h4>第 3–5 分钟：创新设计</h4>
    <p>访问 <a href="/innovation/">AI 教学创新设计</a>，深入审查课前—课中—课后三段式架构、人机协同机制与教学设计理念。</p>
  </div>
  <div class="timeline-item">
    <h4>第 6–8 分钟：案例实证</h4>
    <p>访问 <a href="/cases/lesson-15/">核心案例：第15讲</a>，审查问题工程设计、课堂执行脚本、AI 审查过程与验证路径的完整性。</p>
  </div>
  <div class="timeline-item">
    <h4>第 9–11 分钟：成效验证</h4>
    <p>访问 <a href="/results/">成效与数据</a>，审查对比实验设计、统计方法口径、指标变化趋势与证据可追溯性。</p>
  </div>
  <div class="timeline-item">
    <h4>第 12–13 分钟：支撑材料</h4>
    <p>访问 <a href="/resources/">教学资源</a>，核查提示词模板、测试清单、文档下载与证据索引是否完整齐备。</p>
  </div>
  <div class="timeline-item">
    <h4>第 14–15 分钟：推广价值</h4>
    <p>访问 <a href="/promotion/">推广与复用</a>，评估实施条件、迁移方法与跨课程复用的可行性与推广潜力。</p>
  </div>
</div>

## 常见评审问题

<div class="info-box blue">
  <span class="icon">❓</span>
  <div>
    <strong>AI 是否只是简单的工具替代？</strong><br>
    不是。本项目中 AI 承担了任务生成、代码审查、数据评价三重角色，深度嵌入课前—课中—课后全流程，而非仅用于演示或辅助搜索。详见 <a href="/innovation/">AI 教学创新设计</a> 页面的人机协同架构。
  </div>
</div>

<div class="info-box emerald">
  <span class="icon">❓</span>
  <div>
    <strong>成效数据的统计方法是否可靠？</strong><br>
    所有量化指标均标注了数据来源、采集时间、样本量与统计方法。对比实验采用同一教师、同一课程、不同学期的准实验设计，控制了主要混淆变量。详见 <a href="/results/">成效与数据</a> 页面。
  </div>
</div>

<div class="info-box amber">
  <span class="icon">❓</span>
  <div>
    <strong>该模式是否依赖特定 AI 工具？</strong><br>
    不依赖。核心设计基于提示词模板与结构化流程，可适配任何主流生成式 AI 工具。最低实施仅需一个可用的 AI 对话工具，不绑定特定产品或平台。详见 <a href="/promotion/">推广与复用</a> 页面的实施条件。
  </div>
</div>

<div class="info-box blue">
  <span class="icon">❓</span>
  <div>
    <strong>如何验证课堂实施的真实性？</strong><br>
    核心案例（第 15 讲）提供了完整的课堂执行脚本、时间轴记录、学生代码提交截图与 AI 审查输出记录。所有证据均可在 <a href="/cases/lesson-15/">核心案例</a> 与 <a href="/resources/">教学资源</a> 页面交叉验证。
  </div>
</div>
