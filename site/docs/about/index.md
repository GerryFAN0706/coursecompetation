# 关于

## 项目背景

本网站服务于 `Java Web应用开发` 课程的人工智能教学创新参赛展示，目标是将课堂改革过程、教学证据与推广价值进行结构化呈现。

## 版本说明

| 版本 | 用途 | 特点 |
|---|---|---|
| Internal（原始版） | 校内存档/复核 | 保留完整原始证据 |
| Public（匿名评审版） | 专家评审 | 去身份信息、保留可验证证据链 |

当前在线版本为 `Public（匿名评审版）`。

## 匿名评审与合规

发布前执行以下检查：

1. 去除姓名、学号、账号标识等敏感信息。
2. 按要求处理学校标识与机构信息。
3. 检查截图中的浏览器与平台残留身份信息。
4. 为每个关键指标补充统计口径与证据来源。

检查清单见 [匿名化检查清单](/anonymity-checklist)。
