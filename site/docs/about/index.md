# 关于本项目

<div class="section-lead">本网站服务于 Java Web应用开发 课程的人工智能教学创新参赛展示，为第六届全国高校教师教学创新大赛（人工智能赛道）提供结构化的教学改革成果呈现。</div>

<div class="summary-grid">

## AI使用规范与学术诚信

<div class="info-box emerald">
  <span class="icon">🛡️</span>
  <div>本项目始终坚持"AI是辅助者，不是替代者"的核心原则。以下是我们在教学中贯彻的AI使用规范。</div>
</div>

<div class="summary-grid">
  <div class="summary-card">
    <div class="summary-icon">🚫</div>
    <h3>AI不做什么</h3>
    <p>不直接给出修复代码、不替代学生思考、不自动评分定等级、不替代教师教学判断</p>
  </div>
  <div class="summary-card">
    <div class="summary-icon">✅</div>
    <h3>AI做什么</h3>
    <p>辅助诊断代码缺陷、生成差异化任务、输出结构化反馈、提取班级共性问题</p>
  </div>
  <div class="summary-card">
    <div class="summary-icon">👨‍🏫</div>
    <h3>教师做什么</h3>
    <p>设计提示词边界、审核AI输出、纠偏错误诊断、把控评价标准、引导知识迁移</p>
  </div>
  <div class="summary-card">
    <div class="summary-icon">👩‍🎓</div>
    <h3>学生做什么</h3>
    <p>使用AI辅助发现问题、独立判断AI建议、编写修复代码、解释修改原因、完成验证</p>
  </div>
</div>

### 防滥用措施

<div class="flow-grid">
  <div class="flow-card">
    <div class="step-num">1</div>
    <h4>提示词边界控制</h4>
    <p>AI只输出问题位置和风险说明，不给完整修复代码</p>
  </div>
  <div class="flow-card">
    <div class="step-num">2</div>
    <h4>教师审核机制</h4>
    <p>AI输出经教师确认后才作为教学依据，纠偏错误诊断</p>
  </div>
  <div class="flow-card">
    <div class="step-num">3</div>
    <h4>一人一题机制</h4>
    <p>不同学生不同业务场景，AI生成答案无法互相复用</p>
  </div>
  <div class="flow-card">
    <div class="step-num">4</div>
    <h4>知识验证闭环</h4>
    <p>学生需解释修改原因，教师追问知识点，测试清单验证</p>
  </div>
</div>
